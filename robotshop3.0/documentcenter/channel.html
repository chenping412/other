资讯频道

接口代码:collector.channel.getInfos

所需data参数：

参数 必输 类型 备注
channelCode 是 string 关键词
op 否 string 不填时默认为0，当op为1时，表示获取历史数据，t传上一批返回数据中最后一条的t
t 否 string 不填时默认为0,代表取最新数据;t 不为0时，代表查看以前的数据

返回参数说明：

字段名 数据类型 备注
code 字符串 返回码 0：成功 其他则表示失败
msg 字符串 提示信息
data 数组 业务参数
_id 字符串 新闻id
title 字符串 资讯标题
time 字符串 发布时间
site 字符串 来源
image 数组 图片
tags 数组 新闻标签
detailUrl 字符串 详情页地址，uid参数需要客户填写
url 字符串 原文链接
_tid 字符串 tid
