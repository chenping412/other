资讯追踪

1添加关键词

接口代码：collector.track.add

所需data参数：

参数 必输 类型 备注
keyword 是 string 关键词

返回数据JSON示例：

{
    "code": "0",
    "msg": null,
    "data": {
        id=9, 
        addResult=success
    }
}

返回参数说明：

参数 类型 备注
2 删除关键词

接口代码：collector.track.del

所需data参数：

参数 必输 类型 备注
keyword 是 string 关键词

返回数据JSON示例：

{
    "code": "0",
    "msg": null,
    "data": {
        delResult=success
    }
}

返回参数说明：

参数 类型 备注
delResult string 删除成功标记，success表示成功，其他表示失败
3 追踪词列表

接口代码：collector.track.list

所需data参数：

参数 必输 类型 备注
pageNo 否 string 页码，默认为1
pageSize 否 string 每页展示数量，默认为10

返回数据JSON示例：

{
    "code":"0",
    "msg":null,
    "data":[
        {
            "id":9,
            "keyword":"区块链投资",
            "username":"19"
        }
    ]
}

返回参数说明：

参数 类型 备注
keyword string 追踪词
username string 用户账号信息
id string 追踪词id
