接入说明 资讯搜索
  • 1、api说明

    接口访问前缀：http://robotapi.giiso.com/openapi/

  • 2、账号信息
    字段 字段 字段
    username API密钥内的SecretId 用户认证名，公共参数中用来验证身份
    secretKey API密钥内的Secretkey 用户认证秘钥，公共参数中用来验证身份
    customId 注册手机号 客户id
  • 3、公共参数说明

    请求头参数：

    字段 数据类型 必填 字段
    date 字符串 是 格林威治时间，格式：EEE, dd MMM yyyy HH:mm:ss z
    Authorization 字符串 是 用户认证秘钥，公共参数中用来验证身份

    Authorization各参数说明：

    字段 数据类型 必填 备注
    username 字符串 是 用户凭证
    algorithm 字符串 是 签名算法，固定值：hmac-sha1
    headers 字符串 是 参与签名的HTTP请求头列表，暂只填固定值：date
    signature 字符串 是 Base64编码的签名串

    如用户凭证为“bob” ，则最终Authorization请求头格式为：

    hmac username="bob", algorithm="hmac-sha1", headers="date", signature="Base64(HMAC-SHA1(signing string))"
    待签名字符串（即上面的signing string）为：date: Fri, 09 Oct 2015 00:00:00 GMT
    signature计算方式：先用secretKey初始化签名算法，再用HMAC-SHA1对签名字符串进行加密，然后对加密后的内容进行base64编码
    初始化签名算法java示例：
    Mac hmacSha1 = Mac.getInstance(AUTH_TYPE);
    byte[] keyBytes = secretKey.getBytes();
    hmacSha1.init(new SecretKeySpec(keyBytes, AUTH_TYPE));

    • C#请参照GiisoAPI-demo-c#.rar 下载

开始使用 接口调用方法 接口说明文档

一. 开始使用

  • 1.1，引入js文件

    在页面中引入GRobot-JS-SDK.min.js文件，如下：

    <script type="text/javascript" src="http://robotsdk.giiso.com/SDK/js/GRobot-JS-SDK.min.js?se
cretId=C0001160003&secretKey=ca56e5ef-9b3c-43e5-8b42-e8672d2d8ee8">
  • 1.2，初始化GRobot实例

    在使用数据接口之前，必须调用GRobot.ready()方法，进行初始化操作

    GRobot.ready(function(){
    console.log('ready')
})

二. 接口调用方法

  • 2.1 ，调用接口的方法

    在使用数据接口之前，必须调用GRobot.ready()方法，进行初始化操作

    GRobot.getData({
    apiName: 'collector.search',

    data: {
        keyword: '王宝强'
    },

    success: function(data){
        console.log(data)
    },

    error: function(status){
        console.log(status)
    },

})
  • 传入字段的说明
    字段 是否必传 说明
    apiName 是 字符串，要调用的接口名称，务必填写正确（中间包含 符号 . 不可省略）
    data 否 接口请求参数，具体请看对应接口文档，如果接口无需参数，该字段可以不传
    success 是 成功回调，请求成功时回调函数，data为成功后接口返回数据，JSON对象
    error 否 失败回调，请求失败时回调函数，status为错误状态码
